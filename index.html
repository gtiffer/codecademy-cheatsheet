HTML Table CSS

HTML References

TAG NAME DESCRIPTION
<li> List Item Element Create list items inside ol and ul.
<video> Video Element embeds a media player for video playback. The src attribute will contain the URL to the video. Adding the controls attribute will display video controls in the media player.
<em> Emphasis Element Emphasizes text and browsers will usually italicize the emphasized text by default.
<ol> Ordered List Element Creates a list of items in sequential order.
<div> Div Element used as a container that divides an HTML document into sections and is short for “division”.
<br> Line Break Element will create a line break in text and is especially useful where a division of text is required, like in a postal address.
<img> Image Element Embed images in documents. The src attribute contains the image URL and is mandatory.
<p> Paragraph Element Contains and displays a block of text.
<ul> Unordered List Element Used to create a list of items in no particular order.
<body> Body Element Represents the content of an HTML document. Content inside body tags are rendered on the web browsers. Note: There can be only one body element in a document.
<span> Span Element An inline container for text and can be used to group text for styling purposes.
<strong> Strong Element Highlights important, serious, or urgent text and browsers will normally render this highlighted text in bold by default.

Table References

TAG NAME DESCRIPTION
<table> Table Element Has content that is used to represent a two-dimensional table made of rows and columns.
<tr> Table Row Element Used to add rows to a table before adding table data and table headings.
<td> Table Data Element Can be nested inside a table row element to add a cell of data to a table.
<thead> Table Head Element Defines the headings of table columns encapsulated in table rows.
<tbody> Table Body Element A semantic element that will contain all table data other than table heading and table footer content.

CSS References

TAG NAME DESCRIPTION
<link> Link Element Used to link HTML documents to external resources like CSS files
href URL Attribute Specify the URL to the external resource
rel Relationship Attribute specify the relationship of the linked document to the current document.
type Type Attribute Defines the type of content being linked.
id = "identify" #identify ID Identifier CSS ID selectors should be unique and used to style only a single element. ID selectors are denoted with a hash sign # followed by the id name.
class = "identify" .identify Class Identifier classes can be reusable and applied to many elements. Class selectors are denoted with a period . followed by the class name.
font-size Font Size Used to set text sizes. Font size values can be many different units or types such as pixels.
background-color Background Color Controls the background color of elements.
!important Important Rule Used on declarations to override any other declarations for a property and ignore selector specificity.
opacity Opacity Used to control the transparency of an element. The value of this property ranges from 0 (transparent) to 1 (opaque).
font-weight Font Weight Property can be used to set the weight (boldness) of text. The provided value can be a keyword such as bold or normal.
text-align Text Align Can be used to set the text alignment of inline contents. This property can be set to these values: left, right, or center.
color Text Color Foreground text color of an element can be set in CSS.
background-image Background Image CSS property sets the background image of an element. An image URL should be provided in the syntax url("moon.jpg") as the value of the property.
font-family Font Family Used to specify the typeface in a rule set. Fonts must be available to the browser to display correctly, either on the computer or linked as a web font.
